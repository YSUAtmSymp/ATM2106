How to read files and plot variables

Date Mon 05 March 2018 Tags python

How to read a data file and make a plot.¶

In this example, we try to download the data file, read it and make a plot.
First, we need some packages to enable us to do what we want to.
The package called numpy allows us to do a wide range of things, and it is essential in most of the time.
Another package that we will use a lot is matplotlib that allows us to make some plots.

We cam load these two packages using the command import.

In [1]:
import numpy as np
import matplotlib.pyplot as plt

Now, let's download the data file to our folder.
In this case, we save SWR_jra55.npz as SWR_jra55.npz

Okay, it is there. Now, let's read the data file using np.load command.

In [4]:
data = np.load('SWR_jra55.npz')

data contains several fields. To check it, we do data.keys().

In [5]:
data.keys()
Out[5]:
['lat', 'SWR', 'lon']

Let's assign individual name for each field.

In [6]:
lat = data['lat']
lon = data['lon']
SWR = data['SWR']

As you will see, they are numpy arrays, meaning they are a structured group of numbers.
To see the structure, you can try shape function.

In [7]:
print np.shape(lon), np.shape(lat), np.shape(SWR)

(145, 288) (145, 288) (12, 145, 288)

We can get the same result with .shape

In [8]:
print lon.shape, lat.shape, SWR.shape

(145, 288) (145, 288) (12, 145, 288)

The numbers above indicate the size of the arrays. We can see that lon and lat are 2-dimensional arrays, while SWR is a 3-dimensional array.
Two dimensions in lon and lat are in x- and y-directions, and additional dimension in SWR represents time in this case (I know this because I structured it.)

Plotting¶

Now, we are almost ready to make a plot for the incoming solar shortwave radiation (SWR).
We will use matplotlib package for the plot.

The first step is to define the size of the canvas for our plot.
Then show SWR in January with colors.

In [9]:
# This allows us to see the figure in the same browser.
%matplotlib inline    

plt.figure(figsize=(16,8))
plt.pcolormesh(lon, lat, SWR[0, :, :])
plt.colorbar()
Out[9]:

We can adjust our plot by changing color scheme or the range of the values...

In [10]:
plt.figure(figsize=(16,8))    # create a figure
plt.pcolormesh(lon, lat, SWR[0,:,:], cmap='Reds', vmin=0, vmax=400)       # make a plot
plt.title('Incoming solar shortwave radiation, January (W/m$^2$)',\
          fontsize=20)  # title
plt.xlabel('Longitude', fontsize=15)    # x label
plt.ylabel('Latitude', fontsize=15)     # y label
plt.tick_params(labelsize=15)           # make the ticks (numbers on the axes) bigger
plt.colorbar()                          # colorbar

If you mave more than one plot on one figure, we need to make sub-spaces for each plot.

f, ax = plt.subplots(2,1, figsize=(16,16))    # create a figure and prepare 4 spaces

a0=ax[0].pcolormesh(lon, lat, SWR[0,:,:], cmap='jet', vmin=0, vmax=400)
ax[0].set_title('SWR', fontsize=20)
ax[0].set_xlabel('Longitude', fontsize=15)
ax[0].set_ylabel('Latitude', fontsize=15)
ax[0].tick_params(labelsize=15)
plt.colorbar(a0, ax=ax[0])

a1=ax[1].pcolormesh(lon, lat, SWR[7,:,:], vmin=0, vmax=400, cmap='jet')
c1=ax[1].contour(lon, lat, SWR[7,:,:], range(0,401,50), colors='black')
ax[1].clabel(c1, fmt='%3.0f', colors='black')
ax[1].set_title('SWR', fontsize=20)
ax[1].set_xlabel('Longitude', fontsize=15)
ax[1].set_ylabel('Latitude', fontsize=15)
